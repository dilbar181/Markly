# 🚀 Markly — Bookmark Manager

Markly adalah aplikasi **Bookmark Manager modern berbasis web** yang dibuat menggunakan **Vanilla JavaScript (tanpa framework)** dengan desain minimal, clean, dan responsif.

Aplikasi ini memungkinkan kamu untuk menyimpan, mengelola, dan mengorganisasi bookmark dengan mudah menggunakan **tag, pencarian, dan fitur interaktif lainnya**.

---

## ✨ Fitur Utama

* 📌 Tambah, edit, dan hapus bookmark
* 🏷️ Sistem tag (kategori fleksibel)
* 🔍 Pencarian real-time
* 📊 Filter berdasarkan tag
* ↕️ Sorting (Terbaru, Terlama, A-Z, Z-A)
* 🧠 Autocomplete tag suggestion
* ➕ Tambah tag manual
* ❌ Hapus tag dengan konfirmasi
* 🌙 Dark / Light mode
* 💾 Penyimpanan lokal (LocalStorage)
* 📂 Import & Export JSON
* 🖱️ Drag & Drop reorder bookmark
* ⌨️ Keyboard shortcut (Ctrl + K search)
* 🔔 Toast notification
* 📱 Responsive (mobile-friendly)

---

## 🛠️ Teknologi

## 💡 Cara Penggunaan

### ➕ Tambah Bookmark

* Klik tombol **"Tambah Bookmark"**
* Isi judul, URL, dan tag
* Klik **Simpan**

### 🏷️ Tag

* Ketik di input tag → otomatis muncul rekomendasi
* Tekan **Enter** untuk menambahkan
* Klik **+** untuk tambah tag manual

### ❌ Hapus Tag

* Klik icon ❌ di sidebar
* Konfirmasi sebelum menghapus

### 🔍 Search

* Gunakan search bar atau tekan `Ctrl + K`

### 📂 Export / Import

* Export: simpan data ke file JSON
* Import: load kembali bookmark
